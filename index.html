# Ability Tools for Thimble #

Create talking web pages quickly and easily. This Thimble project site
will help you make a web page that reads the contents aloud on a
computer or a supported tablet.

**Teachers**: Check out the [READ
ME](https://thimbleprojects.org/jimholgate/286916/README.html) guide for
tips on sharing class projects.
